# MovieApp


Android პროექტი, რომელიც რეალურ დროში ქაჩავს ფილმების მონაცემებს TMDB API-დან. 
აპლიკაციის მთავარ გვერდზე ეგრევე ჩანს პოპულარული ფილმები, თუმცა შეგიძლია ჩაწერო ფილმის სახელიც და ძებნა იწყება ავტომატურად, წერის პროცესშივე. ფილმზე დაჭერისას გადადის სხვა გვერდზე, სადაც უფრო დიდ პოსტერს, აღწერას და რეიტინგს ნახავ. ფილმებს ბარათებზე ზემოდან აწერია რეალური ქულები, მაგალითად 7.1, რომელიც პირდაპირ API-დან მოდის.
დეტალების ფრაგმენტზე დავამატე ფუნქციური "Back" ღილაკი, რომელიც პოსტერზეა განთავსებული. მასზე დაჭერისას აპლიკაცია იყენებს Navigation Component-ს, რათა მომხმარებელი უსაფრთხოდ და მარტივად დააბრუნოს მთავარ გვერდზე (HomeFragment). ეს ფუნქცია აუმჯობესებს აპლიკაციის ნავიგაციას და მომხმარებლის გამოცდილებას.
ტექნიკური მხარისთვის გამოვიყენე ViewBinding, რათა კოდიდან UI ელემენტებთან მუშაობა უფრო მარტივი ყოფილიყო. ინტერნეტიდან მონაცემების წამოსაღებად გამოვიყენე Retrofit, ხოლო ფილმების პოსტერების გამოსაჩენად Glide ბიბლიოთეკა. გვერდებს შორის გადასასვლელად გამოვიყენე Navigation Component, ხოლო Safe Args-ის დახმარებით ერთი გვერდიდან მეორეზე ფილმის ინფორმაცია უსაფრთხოდ გადადის.
აპლიკაციის ჩართვისას Retrofit უკავშირდება TMDB-ს და მოაქვს ფილმები, რასაც RecyclerView ლამაზად ალაგებს Grid სტილში. თუ ძებნისას რამეს ჩაწერ და შედეგი არ მოიძებნა, აპლიკაცია გეტყვის, რომ "No movies found".
